🏃 LapMaster Pro — спортивный секундомер с кругами

> Профессиональный секундомер для спортсменов с поддержкой кругов, лучшего/худшего времени, статистики и экспорта данных.  
> Реализован на 7 языках программирования для демонстрации кроссплатформенных подходов.

## 🧠 Концепция

**LapMaster Pro** — это не просто секундомер. Это полноценный инструмент для тренировок:

- ✅ **Высокая точность** — до миллисекунд.
- ✅ **Измерение кругов (сплитов)** — запись времени каждого круга.
- ✅ **Автоматический расчёт** лучшего, худшего и среднего времени круга.
- ✅ **Пауза и возобновление** с сохранением прогресса.
- ✅ **Экспорт** данных в CSV/TXT для анализа.
- ✅ **Звуковой сигнал** при каждом круге (опционально).
- ✅ **Визуальный таймер** с крупными цифрами.
- ✅ **Горячие клавиши** для быстрого управления (пробел, Enter и др.).

## 🚀 Функциональность

| Действие                 | Описание |
|--------------------------|----------|
| `Старт`                  | Запуск секундомера. |
| `Стоп`                   | Остановка (можно возобновить). |
| `Круг (Lap)`             | Фиксация времени текущего круга. |
| `Сброс`                  | Сброс всех показаний и кругов. |
| `Показать круги`         | Список всех кругов с номерами и временем. |
| `Лучший круг`            | Отображение наименьшего времени. |
| `Худший круг`            | Отображение наибольшего времени. |
| `Среднее время`          | Расчёт среднего арифметического всех кругов. |
| `Экспорт в CSV`          | Сохранение кругов в файл для дальнейшего анализа. |

## 📦 Содержимое репозитория

| Файл                  | Язык       | Особенности |
|-----------------------|------------|-------------|
| `stopwatch_python.py` | Python     | + Tkinter GUI, звук через `playsound`, экспорт CSV |
| `stopwatch_cpp.cpp`   | C++        | + Qt Widgets, QSound, таблица кругов |
| `stopwatch_java.java` | Java       | + Swing, JTable, экспорт через `FileWriter` |
| `stopwatch_cs.cs`     | C#         | + WPF, DataGrid, звук через `System.Media` |
| `stopwatch_go.go`     | Go         | + консоль с цветами, команды, запись в файл |
| `stopwatch_rs.rs`     | Rust       | + консоль, `termion` для цветов, `chrono` для времени |
| `stopwatch_js.js`     | JavaScript | + Electron, React-подобный интерфейс (чистый HTML) |

## 🛠️ Как запустить

Каждая версия самодостаточна. Установите нужные зависимости и выполните:

```bash
# Python
pip install playsound
python stopwatch_python.py

# C++ (Qt)
qmake && make
./stopwatch_cpp

# Java
javac -cp . stopwatch_java.java && java stopwatch_java

# C# (WPF)
dotnet run

# Go
go run stopwatch_go.go

# Rust (сборка)
cargo build --release && ./target/release/stopwatch_rs

# Node.js (Electron)
npm install
npm start
